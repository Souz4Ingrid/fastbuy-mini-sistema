# FastBuy

Projeto de site de compras com catálogo de jogos, carrinho funcional e modal de login/cadastro.

## Tecnologias

* HTML
* CSS
* JavaScript

## Funcionalidades

* Header com navegação
* Ícone de pesquisa com barra expansível
* Modal de login e cadastro
* Alternância entre formulários
* Contador de itens no carrinho
* Grade de produtos com imagem, título, preço e botão
* Responsividade

## Como usar

1. Clique no ícone de login para abrir o modal.
2. Use os links para alternar entre login e cadastro.
3. Clique em "Adicionar" para aumentar o contador do carrinho.
4. Use o ícone de pesquisa para abrir a barra de busca.

## Como rodar

1. Baixe ou clone o repositório:

   ```bash
   git clone https://github.com/Souz4Ingrid/FastBuy.git
   ```
2. Abra o arquivo `index.html` no navegador.
